```text
2️⃣ Tools Required (Linux Setup for Android)
Tool	Purpose
Python 3	Base language
Kivy	GUI + logic for the app
Buildozer	Convert Python script → Android .apk
Java JDK	Required for Android build tools
Android SDK / NDK	Needed by Buildozer for compilation

Tip: Linux me Buildozer aur Kivy ka combination easiest hai beginners ke liye.
```

```text

3️⃣ Simple Workflow to Create Android App in Python

Install Kivy → GUI + logic likho

Write Python app → buttons, inputs, outputs

Install Buildozer → prepare .spec file

Run Buildozer → Python code ko .apk me convert karo

Install .apk on Android → test app
```
